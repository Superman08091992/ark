### Using the CLI Tool

#### 1. View Statistics

```bash
./bin/ark-lattice stats
```

**Output:**
```
üìä Code Lattice Statistics:

   Languages: 3

```

#### 2. Query Nodes

```bash
# By language
./bin/ark-lattice query --language python --limit 5

# By type
./bin/ark-lattice query --type template_node --limit 10

# By capability
./bin/ark-lattice query --capability "REST API"
```

#### 3. List Nodes

```bash
# List all Python nodes
./bin/ark-lattice list --language python

# List all pattern nodes
./bin/ark-lattice list --type pattern_node

# List first 20 nodes
./bin/ark-lattice list --limit 20
```

#### 4. Generate Code

```bash
# Generate Python code
./bin/ark-lattice generate --requirements "REST API server" --language python

# Generate JavaScript code
./bin/ark-lattice generate --requirements "web server" --language javascript

# Save to file
./bin/ark-lattice generate --requirements "hello world" --language rust --output hello.rs
```

#### 5. Add Custom Node

```bash
./bin/ark-lattice add \
  --id my_custom_node \
  --type template_node \
  --value "My custom template" \
  --language python \
  --capabilities "custom" "template" \
  --content "print('Custom code')"
```

#### 6. Import Nodes from JSON

```bash
./bin/ark-lattice import /path/to/nodes.json
```

#### 7. Export All Nodes

```bash
./bin/ark-lattice export output.json
```

#### 8. Delete a Node

```bash
./bin/ark-lattice delete node_id
```
